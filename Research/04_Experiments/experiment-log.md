# Experiment Log

The dashboard reads structured experiment entries from `data/experiments.json`.

## Experiment Template

```text
# Experiment:

## Question

## Motivation
Why interesting/useful?

## Setup
Model, data, code, config.

## Result

## Interpretation
What does it say? What does it not say?

## Decision
Continue / Change direction / Stop / Write memo

## Links
Repo, wandb, notebook, logs.
```
